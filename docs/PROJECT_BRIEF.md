# Project Brief

## Working Name

Screen Active Task Reminder

## One-Line Description

A lightweight Android app that nudges users to review pending tasks when they spend too long active on their phone.

## Target User

People who already use Google Tasks and want a gentle interruption when phone use becomes distracting.

## Problem

Task apps are passive. Users can have important tasks waiting, but they may continue using the phone without checking them. Existing screen-time tools often block apps or create friction. This app should be lighter: remind, do not punish.

## Product Promise

Help users notice pending work at the right time with minimal setup, minimal battery impact, and a calm modern interface.

## Primary Goals

- Easy first-time setup.
- Fast local experience.
- No required paid infrastructure.
- Configurable reminders.
- Clean Android-native UI.
- Privacy-respecting local-first design.

## Non-Goals For V1

- iOS support.
- Web dashboard.
- Custom backend.
- Team collaboration.
- AI task planning.
- Ads in the core reminder flow.
- Mandatory AccessibilityService monitoring.
- Forced full-screen overlays outside the app.

## Monetization Direction

Start free. Add a one-time Pro unlock after V1 has real user feedback.

Free features:

- Google Tasks read sync.
- Basic pending-task dashboard.
- One reminder interval.
- Basic snooze.
- Light and dark theme.

Potential Pro features:

- Multiple reminder profiles.
- Custom reminder intervals.
- Advanced quiet hours.
- Per-task-list filters.
- Priority-based reminders.
- More notification actions.
- Custom themes.

## Success Metrics

- User completes onboarding in under 2 minutes.
- User grants notification permission.
- First task sync succeeds.
- Reminder fires reliably.
- User returns after 7 days.
- No common complaint about battery drain.
- No common complaint about confusing setup.
